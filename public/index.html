캔버스 자동 분류기

엑셀 파일을 업로드하면 자동으로 분류 결과 파일을 내려받습니다.
드래그&드롭도 지원합니다.

파일을 여기에 놓거나, 버튼으로 선택하세요

권장: .xlsx / .xls
(파일 선택 창에서 모든 파일(*.*) 선택 가능)

선택된 파일 없음
대기 파일을 선택하면 자동 다운로드됩니다.
서버로 업로드 후 결과 xlsx를 반환합니다.
© canvas-classifier 문제가 있으면 새로고침 후 다시 시도하세요.

사용 안내

파일마다 헤더 위치가 달라도 자동으로 탐지합니다.

지원 기능

  • 멀티 시트 처리 (sheet_result + sheet_summary)
  • 상품명(분류) / 아이템 코드 / 설명 / 수량 / 박스수 출력
  • 요약줄(Grimso 등)은 상품명 출력 안 함
  • 엑셀 스타일(테두리/헤더 굵게) 적용
업로드 엔드포인트 /upload
출력 파일 _분류결과.xlsx
권장 브라우저 Chrome/Edge
팁: 파일명이 길면 일부만 표시됩니다. 업로드 전에 파일명 확인 후 진행하세요.
